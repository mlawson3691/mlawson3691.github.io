Leaf, Root, & Petal

[view live site] [view github repo]

This application is an eCommerce site representing a personal business idea selling natural, homemade soaps and cosmetics. Administrators can upload products to the site and post articles to the blog, while other users can browse the store and blog, add items to their cart, and checkout.

Technologies:

  • PHP
  • Drupal 7
  • Drupal Commerce module
  • Bootstrap sub-theme

Geography Quiz

[view live site] [view github repo]

This application serves as a geography quiz game where the computer names a country and the user must pinpoint that country on the map. Once a user guesses wrong, their score resets.

Technologies:

  • JavaScript
  • Node.js
  • Gulp
  • Google Maps API
